# Cellphone Price Prediction App

This project predicts mobile phone prices based on hardware specifications using Python. The project is implemented entirely in a **Jupyter Notebook** for data analysis, preprocessing, and modeling.

## Features
- Exploratory Data Analysis (EDA) on mobile phone dataset
- Data cleaning and preprocessing
- Predict price based on weighted features:
  - RAM
  - CPU frequency
  - PPI & Screen Resolution
  - Battery
  - Rear Camera
  - Internal Storage
- Visualizations:
  - Sample dataset preview
  - Price distribution histogram
  - Feature vs Price scatter plots

## Technologies
- Python 3
- pandas, numpy
- matplotlib, seaborn
- Jupyter Notebook
